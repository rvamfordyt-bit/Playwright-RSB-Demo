# Playwright-RSB-Demo
    Testing Scope
        Valid Login
        Invalid Login
    How to install
        Install Git
        Install Node
        Install Playwright
    How to run
        Type "npx playwright test --ui" in terminal
        Navigate to Test Script with name RSB-LoginSample.spec.js
        Click Run
    How to veiw reports    
        Navigate to Playwright-report folder
        Open the index.html on local file manager
        Select index.html and open the index.html on the browser
